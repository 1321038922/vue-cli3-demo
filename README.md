# 1. webpack4_cli3_demo

## 1.1. Project setup
```
npm install
```

### 1.1.1. Compiles and hot-reloads for development
```
npm run d + projectName
```

### 1.1.2. Compiles and minifies for production
```
npm run build
```

### 1.1.3. Run your tests
```
npm run test
```

### 1.1.4. Lints and fixes files
```
npm run lint
```
